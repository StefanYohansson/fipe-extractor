FIPE Extractor
===

Ferramenta para exportar a tabela fipe do site: http://veiculos.fipe.org.br/

## Requerimentos

* Perl
* Node

## Instalando


1. Perl JSON + Data Dumper

```
sudo cpan /data-dumper-names/
sudo cpan JSON
```

2. Node v5+ (use nvm)

3. Fe

## Como rodar

```
chmod +x extract-fipe.sh
./extract-fipe.sh
```
